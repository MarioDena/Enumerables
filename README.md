# Enumerables
Enumerables project for Microverse

Simple implementation of the built-in methods for enumerables

Including:

* each
* each_with_index
* select
* all?
* any?
* none?
* count
* map
* inject

Under `Testing suite` you will find some general testing calls freely customizable
